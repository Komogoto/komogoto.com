# komogoto.com

Site web de l'association [Komogoto](https://komogoto.com) — Association loi 1901 fondée en 2018.

## À propos

Komogoto opère des services numériques libres, gratuits et hébergés en Europe, accessibles à tous sans conditions. Pas de publicité, pas de données revendues.

**RNA :** W313031584

## Services

| Service | Description |
|---|---|
| [Serveurs Miroirs](https://mirrors.komogoto.com/) | Distribution open source via HTTPS/rsync/FTP à 1 Gbps (F-Droid, Linux Mint, VideoLAN, OPNsense…) |
| [NTP Pool](https://www.ntppool.org/a/komogoto) | Synchronisation temporelle pour 6M+ systèmes dans le monde |
| [Komogoto Analytics](https://umami.komogoto.com/) | Alternative éthique à Google Analytics, basée sur Umami |
| [TechBag.app](https://techbag.app) | Boîte à outils pour les professionnels IT |
| Hébergement Web | Hébergement gratuit pour les membres de l'association |
